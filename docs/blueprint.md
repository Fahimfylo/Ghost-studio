# **App Name**: Nexlify Studio

## Core Features:

- Bento-Style Service Directory: An interactive, responsive grid highlighting full-stack services including ERP development and SaaS architectural solutions.
- Technical Case Study Gallery: A high-performance portfolio layout featuring project descriptions, tech stack pill tags, and dynamic case study views.
- Project Blueprint Generator: An AI-powered tool that analyzes project ideas to suggest optimal database architectures and feature roadmaps based on industry standards.
- Enterprise Lead Intake: A validated contact system utilizing react-hook-form and Zod to capture detailed project requirements and budgets.
- Modular Lead Storage: Data persistence layer using an external database to manage and track inquiries from enterprise product teams.
- Delivery Roadmap Visualization: A step-by-step horizontal timeline demonstrating the agency’s technical process from discovery to deployment.
- Live Client Testimony Ticker: An auto-scrolling infinite marquee showcasing validated client social proof with hover-to-pause functionality.

## Style Guidelines:

- Primary Color: Vivid tech blue (#3B82F6) chosen for its high contrast in dark mode and representation of digital infrastructure.
- Background Color: Deep obsidian grey (#0A0A0B), heavily desaturated to create a premium, focus-heavy agency environment.
- Accent Color: Vibrant electric purple (#8B5CF6) providing an analogous, innovative contrast for call-to-action gradients.
- Recommended font: 'Inter', a neutral grotesque-style sans-serif selected for its machined look and excellent readability in technical contexts. Note: currently only Google Fonts are supported.
- Consistent 24px Lucide React line-icons in primary accent blue for sharp, minimalist technical navigation.
- Highly structured bento-box architecture with precise spacing and defined borders to mimic modern ERP and dashboard systems.
- Smooth fade-up transitions and background radial gradient drifting to create an atmosphere of scalability and technical fluidity.
